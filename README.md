<h1 align="center">Clozer React + Typescript + Tailwind Template ⏪</h1>

<br />

This is a boilerplate template for my personal React projects.

## ✨ Features

-   Built with TypeScript.
-   Routing with React Router DOM
-   📺 Presentation:
    -   ReactJS.
-   🧪 Style:
    -   Tailwind CSS.

## ⌚ Prerequisites

To run this template make sure you have installed all of the following prerequisites on your machine:

-   Node.js.
-   TypeScript.
-   ESLint.
-   Prettier.
-   NPM.

## 💻 Install

```bash
# Environment Variables
cp '.env.example' '.env' # Development

# Install dependencies
npm install
```

## 🚀 Running

```bash
# Run in development mode
npm run dev

# Build for production mode
npm run build
```
